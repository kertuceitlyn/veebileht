GIT JA VSCODE

Soovid koostada veebilehte tiimina? Pole probleemi!

GitHub
Visual Studio Code

1. Mine GitHub-i, lae alla Git ja Visual Studio Code(VSCode)

2. Loo repository (Lisa tiimikaaslased kollaboraatoriks)

3.Klooni oma repositoorium(kopeeri repositooriumi https link) git clone <kopeeritud link>

4. Ava projekt VSCodes -> loo uus fail

5. Peale loomist tuleb fail ka repositooriumisse üles panna.
Kõigepealt mine oma kloonitud projekti kausta käsuga cd, nüüd kasuta käske:
git add. -> git commit -m "mingi sõnum" -> git push (KULDNE TRIO)
