SAMM 2: CSS


STRUKTUUR:

HTMLis loodud struktuuri alusel lood kõigepealt, mida muutma hakkad ning lisad {}
näiteks: body {...}, h1{...}, .table {...}
... asemele lähevad:

VÄRVID:

color, background-color
näidisväärtused: color:red; background-color:#f0f0f0;

MÕÕTÜHIKUD JA PAIGUTUS:

width, margin, padding, position
näidisväärtused: width:80%; margin:0 auto; padding: 10px; position relative;

TÜPOGRAAFIA:

font-family, font-size, line-height
näidisväärtused: font-family: 'Helvetica Neue', Arial, sans-serif; font-size: 16px; line-height: 1.5;

ANIMATSIOONID:

opacity, transition
näidisväärtused: opacity: 0; transition: opacity is ease-in-out;

MOODULID

nt. Flexbox, Grid
display, justify-content (br)näidisväärtused: display: flex; justify-content: space-between;

SAMM 3: JavaScript


STRUKTUUR:

JavaScripti kood saab lisada HTML-dokumendisse,
kas otse HTML-faili <script> elementi abil või eraldi JavaScripti-failina,
Näiteks: <script src="minujavaskript.js" defer></script>
või <script>(siin vahel on JavaScript)</script>

MUUTUJAD JA ANDMETÜÜBID:

Muutujate deklareerimiseks JavaScriptis on "var", "let", "const"
Näiteks: var muutuja = "Tere maailm"; let arv = 26;
const e = 2.71828

FUNKTSIOONID

Funktioonide deklareerimiseks kasutatake JavaScriptis "function"
Näiteks: function hommik(nimi) {
console.log("Tere hommikust, " + nimi + "!");
}

TINGIMUSLAUSED JA LOOPS:

JavaScriptis saab kasutada "if", "else if" ja "else" tingimuslausetega ning "for", "while" ja "do-while" tsükleid.
Näiteks: if (arv > 0) {
console.log("Arv on positiivne");
} else if (arv < 0) {
console.log("Arv on negatiivne");
}

SÜNDMUSED:

JavaScript võimaldab lisada sündmusi HTML-elementidele
Näiteks: võimalik teha nupp, mis salvestab su vajutuse ning teeb midagi selle tulemusena
